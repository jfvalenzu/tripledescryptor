# tripledescryptor
Simple Java utility for encrypt text strings using Triple DES. 

# Requirements

* Maven 2 or higger
* It requires Java 8 or higger


# Running
USAGE: ./tripledes [text_to_encrypt] [keyfile.bg] [vectorfile.bg]

### Compile Instructions

For compile execute: 
```sh
$ mvn clean compile
$ mvn package
```
That's all!!
